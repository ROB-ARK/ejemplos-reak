# Ejercicios Frontend react

Este repositorio contiene una colección de ejemplos prácticos y ejercicios desarrollados con **React**, enfocados en aprender los fundamentos y conceptos avanzados del ecosistema.

## utiizamos reak para poder diseñar las interfaces al igual que css y js


##  Notas
Este proyecto es parte de un proceso de aprendizaje continuo sobre desarrollo Frontend.

## Instalación y Uso

1. Clona el repositorio:
   git clone https://github.com/ROB-ARK/ejemplos-reak.git

2. Entra a la carpeta del proyecto:
   cd nombre-app

3. Instala las dependencias:
   npm install

4. Inicia el servidor de desarrollo:
   npm run dev
